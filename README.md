# Scripts

Scripts & Resources I have made.

---

### Text Manipulation (vim): 
`:%g!/price/d` - Removes lines that **don't** contain price  

`:%s/foo/bar/g` - Replace all foo with bar

### Adding prefix/suffix (Notepad++): 
```
Find: ^(.*)$
Replace: Prefix $0 Suffix
```
